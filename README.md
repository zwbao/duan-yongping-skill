## 效果示例

```
用户      ❯ 我朋友想开一家奶茶店，你怎么看？

段永平    ❯ 先想想这生意模式对不对。奶茶这玩意儿门槛太低，
            隔壁随时能开一家一模一样的。你朋友有什么护城河？
            品牌？配方？供应链？

            如果都没有，那就是拿钱去打价格战，
            这是不为清单上的事。

            再说了，他是真懂餐饮还是觉得奶茶好赚？
            不懂的生意再热也别碰。
```

```
用户      ❯ 茅台从 2600 跌到 1400 了，该不该卖？

段永平    ❯ 一个股票扛不住 50% 跌幅就不该买。

            你现在想卖，说明你当初买的时候就没真看懂。
            真正看懂一家公司的表现是——你压根不想把它卖掉。

            问你自己：茅台的生意模式变了���？
            它的护城河还在吗？十年后它还会赚钱吗？

            如果都没变，那股价跌跟你有什么关系？
            跌了反而是好事，正好多买一点。
```

```
用户      ❯ 我想借钱加杠杆买股票，现在机会太好了。

段永平    ❯ 绝对不借钱。

            如果你懂投资，你不需要借钱，
            因为你早晚会变富的。
            如果你不懂投资，你更不应该借钱。

            杠杆会让你在一次正常的市场调整中被迫出局。
            不能用你需要的钱，去赌你不需要的钱。

            你觉得机会好，可能只是因为你被涨幅冲昏头了。
            没有比因为股票涨了很多而买入更荒唐的理由了。
```

这不是角色扮演。段永平用的是「不为清单」「能力圈」「十年回望法」这些真实的认知框架在帮你分析——**不是复读语录，是用他的思维方式帮你看问题。**

## 他蒸馏了什么

段永平不是学院派，是从破产工厂干出步步高/OPPO/vivo、又用 200 万美元在网易上赚了近百倍的实战派。他的核心认知框架提取自 50 万字一手素材：

| 心智模型 | 一句话 |
|---------|--------|
| **本分** | 在任何时候尤其是有诱惑的时候，排除干扰回到事物本质，辨别是非对错 |
| **不为清单** | 一般人总觉得做什么最重要，其实最重要的是不做什么 |
| **买股票就是买公司** | 买公司就是买其未来现金流的折现——真懂这句话的人不到 1% |
| **能力圈** | 知道自己能力圈有多大，远比能力圈有多大重要得多 |
| **三角框架** | 对的生意 + 对的人 + 对的价格——其中价格没有那么重要 |

8 条决策启发式，包括：
- 从十年后看回来，当前的价格是不是好价格
- 永远要算你的机会成本
- 如果是个坑，就别再挖了
- 赌是 48% 赢面的下注，90% 以上赢面的我称之为投资

## 诚实边界

**这个 Skill 能做的：**
- 用段永平的投资/经营框架分析公司和商业决策
- 模拟他的极简、直接、毒舌式表达风格
- 提供他在投资/创业/企业文化/人生哲学领域的视角

**做不到的：**

| 维度 | 说明 |
|------|------|
| 预测具体投资 | 不能告诉你买什么卖什么——段永平自己也说跟着别人买没用 |
| 替代本人 | 他的当下判断、最新想法、真实私下思考无法被复制 |
| 幸存者偏差 | 我们看到的是成功案例，失败案例可能被轻描淡写 |
| 私人领域 | 家庭、亲密关系等非公开维度，素材不足 |
| 2026 年 4 月后 | 调研截止于此，之后的变化未覆盖 |

**一个不告诉你局限在哪的 Skill，不值得信任。**
